# 善衡菜单同步审核

生成日期：2026-07-16（Asia/Hong_Kong）

来源：善衡 Order.Place 页面实际请求的 Aigens 全量菜单 API。响应同时包含
B / L / T / D 等时段与稳定 `backendId`，不需要按当前营业时段分别请求。

## 清洗规则

- 每个页面分类只读取首个主商品组；后续商品组是套餐饮品、加购小食、堂食/外卖选项。
- 排除整个「飲品」「零食」「外賣包裝」分类。其中饮品为可乐、水及常见茶餐厅饮品。
- 排除未发布、已归档和 modifier 商品。
- `B` 映射为 `breakfast`，`L` 和下午茶 `T` 映射为 `lunch`，`D` 映射为 `dinner`。
- 外部 key 为 `<backendId>:<mealPeriod>`；同一商品跨午/晚时保留两条菜品记录。
- 上游价格转为 HKD minor units，写入 `pricing.options`；公开菜单暂不显示价格。
