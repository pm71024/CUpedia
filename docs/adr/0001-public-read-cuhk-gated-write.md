# 读公开,写与账号受 CUHK 限制

CUpedia 采用 Wikipedia 式的访问模型:wiki 正文与图片对**匿名用户公开可读**,而注册/登录限 CUHK 邮箱(`@cuhk.edu.hk` 或合规的 `@link.cuhk.edu.hk`),编辑/评论需登录。读与写的边界被有意分离——读公开,写和账号资格受限。

## Considered Options

- **校内 intranet 模型**:正文与图片都要求登录,仅 CUHK 师生可见。注册已限校内邮箱,这是更"自然"的推论。
- **公开阅读模型(选定)**:百科内容面向公众,贡献门槛留给 CUHK 身份。

## Consequences

- 读路径**不得**依赖登录态做访问控制;`getOptionalUser` 仅用于 UI 呈现。
- 账号资格(CUHK 域名白名单)必须在**服务端发码环节**强制,不能只靠客户端——否则公开模型下校外人可绕过白名单注册并编辑。
- 已删除内容是这一公开模型的**例外**:页面正文与历史仅 Admin 可读(见 `CONTEXT.md` 的 Deleted page)。上传媒体不在此例外内——删页不撤图,见 ADR 0002。
